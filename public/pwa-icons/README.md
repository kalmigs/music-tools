PWA icon assets
===============

Current PNGs in this folder were generated from `public/favicon.ico` as placeholders.
For crisp install icons on Android/iOS, replace them with exports from a high-resolution source
(recommended: at least 1024x1024 PNG, no transparency for background-dependent logos).

Required files used by this app:
- `apple-touch-icon.png` (180x180)
- `pwa-192x192.png` (192x192)
- `pwa-512x512.png` (512x512)
- `pwa-maskable-512x512.png` (512x512, safe area centered for maskable icons)
