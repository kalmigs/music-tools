## Getting Started

### Prerequisites

- Node.js 24+
- pnpm 9+

### Installation

```bash
pnpm install
```

### Development

```bash
pnpm dev
```

### Build

```bash
pnpm build
```

### Preview Production Build

```bash
pnpm preview
```

### Format

```bash
pnpm format
```
